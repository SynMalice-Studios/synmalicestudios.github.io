##My personal portfolio website hosted on GitHub Pages with a custom Squarespace domain

## 🚀 Live Demo
You can view the live site here: [https://synmalicestudios.com]

## 🛠️ Built With
* HTML5
* CSS3 (Custom styling and layout)

## 🔧 Deployment & Domain Configuration
This site is automatically deployed using **GitHub Pages**. 
* **Branch:** `main` (or `master`)
* **Custom Domain:** Configured via Squarespace DNS using standard A Records and a CNAME record pointing to `synmalicestudios.github.io`.

⚠️ **Crucial Maintenance Note:** 
Whenever making manual uploads or pushing changes from a local machine, ensure the `CNAME` file containing the custom domain name remains in the root directory. If deleted, GitHub Pages will revert the site back to the default `.github.io` subdomain.

## 📁 File Structure
* `index.html` - The main entry point of the website.
* `background.jpg` - Main background image.
* `CNAME` - Enforces the custom Squarespace domain mapping.
